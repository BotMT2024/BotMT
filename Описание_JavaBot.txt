Описание: бот написан на языке Java версии 18 с использованием фреймворка Spring Boot версии 3.2.5,
для написания функционала бота была использована библиотека  mail-ru-im/bot-java,
в качестве БД был выбран PostgreSQL, в качестве СУБД - pgAdmin
Был реализован следующий функционал:
1) Приветствие юзера
2) Выдача задания для изучения
3) Если юзер в течение часа ничего не напишет после выдачи задания, бот его пингует
4) Интеграция с БД
